# YT-LLM-Summarizer
YT-LLM-Summarizer é uma ferramenta em Python capaz de extrair transcrições de vídeos do YouTube e gerar resumos limpos, estruturados e de alta qualidade usando modelos de linguagem (LLMs). O projeto combina pytubefix, ffmpeg e a API da OpenAI para permitir sumarização rápida de vídeos longos diretamente pelo terminal.
